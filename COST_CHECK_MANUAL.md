このマニュアルに沿って操作すると、今のコストが「無料枠内か」簡単に確認できます。

## 1. Vercel の使用量・コストを確認する

Vercel は、このアプリをインターネット上で動かすためのサーバー（ホスティング）を提供しています。

### ステップ1: Vercel にログインする

1. ブラウザで **https://vercel.com** を開く
2. 右上の **「Log In」** をクリック
3. このアプリをデプロイしたときに使ったアカウント（GitHub など）でログインする

### ステップ2: プロジェクトを選ぶ

1. ログイン後、画面上部に **「Dashboard」** などが表示される
2. **「Projects」** または **「プロジェクト」** をクリック
3. プロジェクト一覧から **「furima-sales」**（またはこのアプリの名前）をクリック

### ステップ3: 使用量を確認する

1. プロジェクトの中に入ったら、左側のメニューを確認する
2. **「Settings」**（設定）をクリック
3. 左メニュー内の **「Usage」**（使用状況）または **「Billing」**（請求）をクリック
4. 画面に表示される内容を確認する：
   - **Bandwidth**（帯域）: 無料枠は約 100GB/月
   - **Serverless Function Invocations**: 無料枠は約 100GB・秒/月
   - **Image Optimization**: 画像最適化の利用量

### ステップ4: プランを確認する

1. 画面上部または右上の **「Upgrade」** 付近を確認する
2. **「Hobby」** と表示されていれば **無料プラン** です
3. **「Pro」** と表示されていれば有料プラン（月額課金）

**確認のポイント**

- 「Hobby」プランで、使用量が赤やオレンジになっていなければ、**基本的に追加料金は発生していません**。
- 「Pro」プランを使っている場合のみ、月額料金がかかります。

## 2. Supabase の使用量・コストを確認する

Supabase は、商品データ・販売データ・画像を保存するためのサービスです。

### ステップ1: Supabase にログインする

1. ブラウザで **https://supabase.com** を開く
2. 右上の **「Sign In」** をクリック
3. このアプリを設定したときに使ったアカウント（GitHub やメール）でログインする

### ステップ2: プロジェクトを開く

1. ログイン後、ダッシュボードが表示される
2. 一覧から **このアプリ用のプロジェクト** をクリック  
   （プロジェクト名は自分で決めたもので、SUPABASE_SETUP.md 内のリンク先も同じプロジェクトです）

### ステップ3: 使用量を確認する（方法A：メニューから）

1. 左側のメニューをスクロールして、一番下付近を確認する
2. **「Project Settings」**（プロジェクト設定）の ⚙️ アイコンをクリック
3. 左メニュー内の **「Usage」**（使用状況）をクリック
4. 画面に次のような項目が表示されます：

| 項目 | 無料枠の目安 | 確認する数値 |
|------|-------------|--------------|
| **Database Size**（データベースの大きさ） | 500MB まで | この数字が 500 以下なら OK |
| **Storage**（ファイル保存容量） | 1GB まで | この数字が 1024（MB）以下なら OK |
| **Bandwidth**（転送量） | 5GB/月 まで | 月内の合計が 5 以下なら OK |

### ステップ4: 使用量を確認する（方法B：メールで）

1. Supabase からの **使用量レポートメール** が届いているか確認する
2. メール内に「You're within your free tier」などと書かれていれば、**無料枠内**です
3. 「Approaching limit」や「Limit exceeded」と書かれていれば、**枠に近いか超えている**状態です

### ステップ5: プランを確認する

1. **「Project Settings」** → **「Billing」** または **「Subscription」** を開く
2. **「Free」** と表示されていれば **無料プラン** です
3. **「Pro」** などと表示されていれば有料プランです

**確認のポイント**

- Database が **500MB 未満**
- Storage が **1GB 未満**
- Bandwidth が **5GB/月 未満**

これらが満たされていれば、**Supabase の追加料金は発生していません**。

## 3. 無料枠を超えたらどうなる？

### Vercel の場合

- **Hobby プラン**：基本的に超過分は使えず、エラーになることがあります
- **Pro プラン**：超過分が従量課金で請求されます
- 対処：使用量が増える場合は、Vercel のダッシュボードでアラートや制限の確認をしておくと安心です

### Supabase の場合

- **Free プラン**：無料枠を超えると、サービスが一時停止されたり、制限がかかることがあります
- 対処：
  - 古い商品画像を削除して Storage を減らす
  - 不要な販売データを整理して Database を減らす
  - 本格運用する場合は **Pro プラン**（有料）への移行を検討する

## よくある質問

**Q: 今すぐお金がかかるか心配です**  
A: 両方とも「Free」や「Hobby」プランで、使用量が赤くなっていなければ、**追加の課金はされません**。まずは上記手順で「Usage」や「Billing」を確認してください。

**Q: どこをクリックすればいいか分からなくなった**  
A: どちらのサービスも「Settings」→「Usage」または「Billing」を開けば、使用量やプランが見られます。左メニューをひとつずつ開いて、該当する項目を探してください。

**Q: 有料プランにしていないのに請求された**  
A: クレジットカードを登録していても、無料枠内であれば請求はされません。請求が来た場合は、各サービスのサポート（Vercel / Supabase のヘルプ）に問い合わせてください。
